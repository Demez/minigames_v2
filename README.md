# Minigames (V2)

### Introduction
The new Minigames gamemode for Garry's Mod. The base has been reworked to make new gamemodes simpler to make!

### Installation
All the folders in this repo should go into garrysmod/gamemodes/

### Gamemodes List
Current gamemodes:
- Balloons
- Ballz
- Bomb Tag
- Crate Wars
- Dodgeball
- Duck Hunt
- Incoming
- Laser Dance
- Melon Racer
- One In The Chamber
- Pitfall
- Poltergeist
